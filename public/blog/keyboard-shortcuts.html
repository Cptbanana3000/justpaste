Handy Keyboard Shortcuts for Faster Note-Taking

Keyboard shortcuts can make writing and editing notes much faster. Try these tips to keep your hands on the keys.

Navigation

  • Use Ctrl/⌘ + F to find text.
  • Use Ctrl/⌘ + K to insert links in some editors.

Formatting

  • Type ## for section headings, then space.
  • Use backticks for inline code and triple backticks for blocks.

Workflow

  • Draft first, format later.
  • Use consistent heading levels for structure.
  • Keep commands and code in fenced blocks.

With practice, you’ll move faster and think less about the UI.
