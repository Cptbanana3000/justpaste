Working Within Note Size Limits (20KB)

Constraints help keep content concise. Here are tricks to stay under the 20KB limit without losing clarity.

Techniques

  • Use headings and bullets over long paragraphs.
  • Link out for large logs or datasets.
  • Prefer code snippets over full files.

Format Efficiently

Markdown is compact, but embedded images or long code can bloat size. Trim carefully.

Iterate

Save the essentials. If the note grows, split it into parts with clear links.

Small notes are easier to read, faster to load, and kinder to bandwidth.
